# MAGA Agency — Landing Page

Landing page for MAGA Agency, a web design & development agency. Built with React 18, TypeScript, Vite, Tailwind CSS, and Framer Motion.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion (scroll-driven animations, reveal-on-scroll)
- lucide-react (icons)

## Getting started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

## Structure

- `src/components/` — page sections (Navbar, Hero, Services, Portfolio, Testimonials, ContactCTA, Footer, BackgroundVideo, LogoMark)
- `src/assets/` — video, images, fonts
- `src/constants.ts` — WhatsApp number, accent color, hero offset
